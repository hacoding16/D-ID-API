# D-ID-API

[![Video Label](http://img.youtube.com/vi/5tdUIkLB51s/0.jpg)](https://youtu.be/5tdUIkLB51s)

### 동영상 ID 생성 API
- URL : https://api.d-id.com/talks

- Header에 API KEY 추가
Key : Authorization
Value : Basic + API KEY

- Body에 CreateId.json 파일의 내용 추가


### 동영상 생성 API
- URL : https://api.d-id.com/talks + 동영상 ID

- Header에 API KEY 추가
Key : Authorization
Value : Basic + API KEY


### 음성 리스트 조회 API
- URL : https://api.d-id.com/tts/voices

- Header에 API KEY 추가
Key : Authorization
Value : Basic + API KEY
